Vs Code

Vs code is an open source code editor developed by Microsoft and also called as IDE(Integrated development Environment) where several tools are inbulit in it. Is is a software where many resourses and tools are used for building realtime projects.

Vs Code Features

  • Inbuilt terminals:In vscode there is an inbuilt terminal or command promt which is used for several tasks like installing modules and performing some programmoing languages and many more.
  • Extensions: There are wide varities of extensions which pushes ahead in development of projects like:
    1. Live server:It refers to an extension or a feature in code editors, like Visual Studio Code, that allows developers to run a local development server with live reload capability.
    2. Live Preview: refers to the feature in development tools and environments that allows developers to see real-time updates of their code's output as they make changes. This feature is particularly useful in web development, design, and content editing
    3. Prettier:It is a popular code formatter that enforces a consistent style by parsing your code and re-printing it with its own rules that take the maximum line length into account, wrapping code when necessary.
    4. Material Icon theme: refers to an icon theme that uses Material Design principles for icons within software applications, particularly in code editors like Visual Studio Code. Material Design is a design language developed by Google
  • Debugging: Debugging Tools: Powerful debugging tools for various languages, including setting breakpoints, inspecting variables, and stepping through code. Debug Extensions: Additional support for different debuggers through extensions.
  • Version Control:Git Integration: Built-in support for Git allows users to manage source code versioning, stage changes, commit, and push from within the editor.
